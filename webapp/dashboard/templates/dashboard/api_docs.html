Overview

The CLEAR25 API provides programmatic access to real-time PM2.5 readings and predictions across Metro Manila. Use it to build integrations, dashboards, or alerts.

Base URL

https://clear25.xyz/api/v1/

Authentication

All API requests require an API key passed in the Authorization header:

Air Quality Levels

The API returns air quality as an integer level (1-5). Here's what each level means:

Level Name PM2.5 Range Health Advisory
1 NONE 0 - 31 µg/m³ No precautions needed.
2 MODERATE 31 - 60 µg/m³ Sensitive groups avoid strenuous activities.
3 HIGH 60 - 80 µg/m³ Everyone reduce physical exertion. Use N95 mask.
4 VERY HIGH 80 - 120 µg/m³ Avoid all outdoor activity. Keep hydrated.
5 EXTREME >120 µg/m³ Halt all indoor pollution sources.

Endpoints

GET /api/v1/live/

Get current PM2.5 readings and 1-hour predictions for all monitoring stations.

Response
{ "stations": [ { "id": "60106", "name": "UST Manila", "city": "Toronto", "lat": 43.6591, "lon": -79.3802, "pm25": 45.2, "predicted": 52.1, "level": 2, "level_name": "MODERATE", "health_advisory": "Sensitive groups avoid strenuous activities." } ], "count": 50, "timestamp": "2025-02-07T10:30:00Z", "age_seconds": 120 }
GET /api/v1/stations/

Get list of all monitoring stations. Optionally filter by city.

Query Parameters
city (optional): Filter by city. Options: {{ cities|join:", " }}
Response
{ "stations": [ { "id": "60106", "name": "UST Manila", "city": "Toronto", "lat": 43.6591, "lon": -79.3802, "tier": 1 } ], "count": 50, "cities": ["Toronto", "Montreal", ...] }
GET /api/v1/cities/

Get list of supported cities with coordinates.

Response
{ "cities": [ { "id": "Toronto", "name": "Toronto", "lat": 43.7479, "lon": -79.2741 } ], "count": 4 }

Rate Limits

The API is rate-limited to ensure fair usage:

  • 100 requests per hour per API key
  • Data refreshes every 30 minutes
  • Responses are cached for 30 seconds

Error Responses

{ "error": "Invalid API key" }

Common HTTP status codes:

  • 401 — Missing or invalid API key
  • 400 — Invalid request parameters
  • 429 — Rate limit exceeded
  • 500 — Server error
